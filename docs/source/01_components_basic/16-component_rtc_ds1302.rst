リアルタイムクロックモジュール (DS1302)
=====================================

.. image:: img/16_DS1302_module.png
    :width: 400
    :align: center

.. raw:: html

   <br/>

DS1302モジュールは、年、月、日、曜日、時、分、秒を追跡できるリアルタイムクロック（RTC）モジュールです。うるう年の調整機能も備えており、正確なタイミングとスケジューリングを必要とするプロジェクトに役立ちます。

仕様
---------------------------
* 供給電圧: 3.3V - 5V
* PCBサイズ: 44 x 23mm
* クロックIC: DS1302
* 動作温度: 0℃ - 70℃

ピン配置
---------------------------
* **VCC**: モジュールの電源供給
* **GND**: グランド
* **CLK**: クロックピン
* **DAT**: データピン
* **RST**: リセットピン
